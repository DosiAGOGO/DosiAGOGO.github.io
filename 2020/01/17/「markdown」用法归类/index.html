「markdown」用法归类

无标签
发布日期:   2020-01-17

网上类似的文章很多，本文为自己对此学习并尝试使用做的一个实践，同时做了一定的整理。

【注】：以下内容的“+”符号均表连接，在实际书写的时候不存在，内容中的“ ”（空格部分）也均不存在，仅仅为了方便查看，需要用空格的地方以文字“空格”表示。

  • 标题

    共有6种等级的标题，写法为：

    【1级标题】 #+空格+标题名
    【2级标题】 ##+空格+标题名
    【3级标题】 ###+空格+标题名
    ……
    【6级标题】######+空格+标题名
    注意：# 与 标题 之间必须跟一个空格，符号才会生效

  • 正文

    【粗体】** + 正文内容 + **
    【展示】加粗样式

    • 【注意】*两个星号后没有空格要求，同时可以使用快捷键Ctrl/Command+B 类似于word操作创建加粗形式，

    对选中文本加粗：选中文本 + Ctrl/Command+B
    创建加粗样式：直接 Ctrl/Command+B*

    下同……

    【斜体】 * + 正文内容 + *
    【展示】斜体样式

    • 快捷键：Ctrl/Command + I

      【斜体加粗】 * + 正文内容 + *
      【展示】斜体加粗样式

    • 快捷键：Ctrl/Command + I + B
    • 【注意】 注意插入字体样式的时候，有时候会显示出代码符号而非文本样式，此时在符号后加一个空格使之与其他文本隔开即可。

  • 插入

    • 分割线

      【分割线】 *** 或 —

    • 【注意】 插入分割线以后很容易导致后面的格式错乱，这时候加一个回车（Enter） 对其隔开即可。

    • 图片

    • 列表

      	- + 空格 + 内容
（Tab）+ - + 空格 + 内容

    • 超链接

      [此处替换显示文字]([此处替换显示文字](链接地址)

      百度

      【注意】 “！”跟注括号[]表示图片

      *没有“！”跟[]表示超链接*

    • 表格

      |表头|表头||表头|表头|表头|
|---|:--:|---:|
|内容|内容|内容|
|内容|内容|内容|

第二行分割表头和内容。
- 有一个就行，为了对齐，多加了几个
文字默认居左
-两边加：表示文字居中
-右边加：表示文字居右

    • 引用

      > + 空格 + 内容
(Tab) + > + 空格 + 内容 （引用中的子引用）

    • 代码

      前后两个单行 ``` 包裹住代码内容。

    • 选择框

      • 选择
